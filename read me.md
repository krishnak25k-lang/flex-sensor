ABOUT MY PROJECT:
      This project demonstrates real-time interaction between flex sensors and a 3D environment using Unity.
*Flex sensors detect bending and convert it into analog signals
*A Raspberry Pi Pico reads these signals and sends normalized values to Unity through serial communication.
*Unity receives the data and maps each flex sensor value to the scaling of a corresponding 3D cube.
*As the sensor is flexed, the related cube expands smoothly, and shrinks back when released.
*This system provides an intuitive visualization of sensor data for learning, rehabilitation, and interactive applications.
